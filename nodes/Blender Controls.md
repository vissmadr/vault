# Blender Controls

`~`: view-related radial menu.

`x`: delete
`shift A`: add

`g`: grab (move)
`alt g`: clear location

`s`: scale
`alt s`: clear scale

`r`: rotate
`alt r`: clear rotation

`tab`: toggle edit mode

`shift n`: recalculate normals

`z`: viewport

## Edit Mode

`1`: select mode: vertex
`2`: select mode: edge
`3`: select mode: face

`ctrl r`: loop cut

`e`: extrude
`i`: inset

`g g`: edge slide

`f`: face
