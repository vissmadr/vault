# Blender Controls

`~`: view-related radial menu.

`w`: selection mode
`a`: select all

`x`: delete
`shift A`: add

`g`: grab (move)
`alt g`: clear location

`s`: scale
`alt s`: clear scale

`r`: rotate
`alt r`: clear rotation

`tab`: toggle edit mode

`z`: viewport

`shift d`: duplicate objects

`o`: proportional editing

`ctrl 1 2 3`: subdivision surface
`ctrl a`: apply modifier

`k`: knife

`ctrl j`: join objects

## Edit Mode

`1`: select mode: vertex
`2`: select mode: edge
`3`: select mode: face

`ctrl r`: loop cut

`e`: extrude
`i`: inset

`g g`: edge slide

`f`: face
`j`: join

`m`: merge
`shift n`: recalculate normals

`h`: hide selected
`shift h`: hide unselected
`alt h`: unhide everything

## Inset

`b`: boundary
