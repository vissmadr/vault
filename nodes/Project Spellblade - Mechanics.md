# Project Spellblade - Mechanics

The fundamental gameplay mechanics.

---

## Actions

**`Q`: Attack**: Melee swing. Instant hit at time `0`.
**`W`: Dash**: Quick dash.
**`E`: Toggle**: Special mode toggle.

While Special is toggled:

**`Q`: Special1**
**`W`: Special2**
**`E`: Special3**

States:

- Free
- Dashing
- Casting
- Stunned

## Animations

Animations should only follow the logic.

States:

- Idling
- Running
- Dashing
- Attacking
- DashAttacking
- Casting
- Stunned
